# Description

Backend for Supply Sphere

## Installation

```bash
npm install
```

## Prerequisites

1. Docker
2. Nodejs > 18.x.x

```bash
# start docker
$ docker compose up

# stop docker
$ docker compose down
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
